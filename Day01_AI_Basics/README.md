
# Day 1 – What is AI? ML vs DL

## 🎯 Objective
Understand the difference between **Artificial Intelligence**, **Machine Learning**, and **Deep Learning**, with real-world **Network Operations** examples.

---

## 1️⃣ What is Artificial Intelligence (AI)?
Artificial Intelligence refers to systems that can:
- Learn from data
- Make decisions
- Predict outcomes
- Mimic human intelligence

### Network Example
An AI system that:
- Analyzes network metrics
- Predicts incidents
- Automatically triggers TTM  
is considered **Artificial Intelligence**.

---

## 2️⃣ Machine Learning (ML)
Machine Learning is a subset of AI where systems learn patterns from historical data without explicit rules.

### ML Flow
Historical Network Data → ML Algorithm → Model → Prediction

### Example
Using CPU, memory, packet drops, and incident history to predict device failure.

---

## 3️⃣ Deep Learning (DL)
Deep Learning is a subset of ML that uses **neural networks with multiple layers**.

### When DL is Useful
- Large datasets
- Log files
- Syslogs
- Time-series metrics

### Example
Analyzing raw network logs to detect anomalies invisible to rule-based systems.

---

## 4️⃣ AI vs ML vs DL Relationship

Artificial Intelligence (AI)
└── Machine Learning (ML)
└── Deep Learning (DL)

---

## ✅ Key Takeaways
- AI is the overall concept
- ML learns from structured data
- DL handles complex and unstructured data
- Network automation uses **AI powered by ML/DL**
